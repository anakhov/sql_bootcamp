## Краткое описание

Рассматриваются основы работы с реляционными базами данных и язык SQL. Изучаются базовые конструкции SQL для выборки данных через серию практических упражнений.

**Основные темы:**
- Написание SELECT-запросов с фильтрацией данных (WHERE)
- Использование операторов сравнения и BETWEEN
- Работа с датами и уникальными значениями (DISTINCT)
- Конкатенация строк и форматирование вывода
- Применение подзапросов в SELECT и FROM clauses
- Использование условных конструкций (CASE)
- Фильтрация по четным идентификаторам
- Сортировка результатов (ORDER BY)

Все задания выполняются на примере базы данных пиццерий, содержащей информацию о посетителях, заказах и визитах.
