## Краткое описание

Изучение процедурных возможностей PostgreSQL, включая триггеры, функции и хранимые процедуры. Рассматриваются различные подходы к реализации бизнес-логики на уровне базы данных.

**Основные темы:**
- Создание системы аудита изменений через триггеры
- Обработка событий INSERT, UPDATE, DELETE
- Объединение логики в универсальные триггеры
- Создание SQL-функций как альтернативы представлениям
- Параметризованные функции с значениями по умолчанию
- PL/pgSQL функции для сложной бизнес-логики
- Работа с массивами и математическими вычислениями
- Реализация алгоритмов (например, Фибоначчи) в функциях

Особое внимание уделяется практическому применению триггеров для отслеживания изменений и созданию гибких функций для работы с данными.
