## Краткое описание

Изучаются операции работы с множествами и JOIN-объединениями в SQL. Рассматриваются различные способы комбинирования данных из нескольких таблиц.

**Основные темы:**
- Операции с множествами: UNION, EXCEPT, INTERSECT
- Работа с дубликатами данных
- Декартово произведение таблиц
- Различные типы JOIN-объединений
- NATURAL JOIN для автоматического соединения
- Сравнение конструкций IN и EXISTS
- Комбинирование данных из нескольких связанных таблиц

Все задания выполняются на примере базы данных пиццерий, с фокусом на объединение данных о посетителях, заказах и визитах.
