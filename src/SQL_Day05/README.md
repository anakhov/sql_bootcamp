## Краткое описание

Изучение создания и оптимизации индексов в PostgreSQL. Рассматриваются различные типы индексов и их влияние на производительность запросов.

**Основные темы:**
- Создание B-Tree индексов для внешних ключей
- Анализ работы индексов через EXPLAIN ANALYZE
- Функциональные индексы для работы с преобразованными данными
- Многоколоночные индексы для сложных условий WHERE
- Уникальные индексы для обеспечения целостности данных
- Частичные индексы для оптимизации работы с подмножествами данных
- Создание покрывающих индексов для ускорения оконных функций

Особое внимание уделяется практическому доказательству эффективности индексов через анализ планов выполнения запросов и метрик производительности.
