## Краткое описание

Погружение в различные типы JOIN-объединений в SQL. Изучаются сложные сценарии работы с данными и продвинутые техники соединения таблиц.

**Основные темы:**
- LEFT/RIGHT/FULL JOIN для работы с отсутствующими данными
- Поиск пропущенных данных и пробелов в датах
- Использование CTE (Common Table Expressions)
- Сложные условия фильтрации в объединениях
- Поиск пересекающихся предпочтений пользователей
- Анализ данных по нескольким критериям одновременно
- Работа с дублирующимися адресами и группировкой людей

Особое внимание уделяется аналитическим запросам для исследования поведения пользователей и их предпочтений в пиццериях.
