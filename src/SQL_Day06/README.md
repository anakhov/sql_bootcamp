## Краткое описание

Расширение бизнес-функциональности базы данных через добавление новых структур и улучшение модели данных. Рассматриваются практические аспекты развития существующей схемы.

**Основные темы:**
- Проектирование и создание новых таблиц для хранения скидок
- Заполнение данных на основе агрегации существующих заказов
- Расчет динамических цен со скидками
- Создание уникальных индексов для предотвращения дубликатов
- Добавление ограничений целостности данных (NOT NULL, CHECK)
- Документирование таблиц и колонок через комментарии
- Использование последовательностей для автоматической генерации первичных ключей

Особое внимание уделяется обеспечению консистентности данных и созданию гибкой системы персональных скидок для клиентов пиццерий.
