## Краткое описание

Изучение операций изменения данных (DML) и продолжение работы с JOIN-объединениями. Рассматриваются практические аспекты модификации данных в базе.

**Основные темы:**
- Сложные аналитические запросы с фильтрацией и сортировкой
- Поиск неиспользуемых данных и пропущенных связей
- Сравнительный анализ посещений и заказов
- Работа с дублирующимися данными и поиск схожих записей
- Операции INSERT для добавления новых данных
- Динамическое вычисление идентификаторов
- Массовые операции обновления и удаления данных
- Практическое применение "insert-select" паттерна

Особое внимание уделяется безопасному изменению данных и отработке сценариев добавления новых пицц, посещений и заказов в базу данных.
