## Краткое описание

Работа с агрегированными данными и аналитическими запросами. Изучаются OLAP-конструкции для получения аналитической информации из сырых данных.

**Основные темы:**
- Базовые агрегатные функции (COUNT, AVG, MAX, MIN)
- Группировка данных и фильтрация групп через HAVING
- Получение уникальных значений и анализ повторяющихся данных
- Комбинирование статистики по посещениям и заказам
- Расчет комплексных метрик для ресторанов
- Анализ географического распределения заказов
- Создание формул на основе агрегированных данных
- Сравнение расчетных показателей со средними значениями

Особое внимание уделяется преобразованию детализированных операционных данных в информативные агрегированные отчеты для бизнес-аналитики.
